# 南瓜棋

> 使用Taro开发一款游戏，支持（微信/百度/支付宝/字节跳动小程序、H5、React-Native） 等，现在在一端端调试。看文件名对应端的代码。

### 运行
```
微信小程序
npm run dev:weapp

h5
npm run dev:h5

其他的
https://nervjs.github.io/taro/docs/GETTING-STARTED.html

```
